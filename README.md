# GermanEd Institute

**Your Digital Pathway to German Universities**

A professional website for GermanEd Institute - a digital language and study academy based in Berlin, Germany, offering online German courses (A1-B1) to prepare students for integration, career, and academic success in Germany.

## About

GermanEd Institute provides high-quality German language education to learners worldwide, with a focus on the Arabic-speaking world, Latin America, and Southern Europe.

### Key Features

- **Online German Courses**: A1, A2, and B1 levels
- **Small Groups**: 8-12 participants for personalized attention
- **Native German Instructors**: Qualified German native speakers
- **Exam Preparation**: Targeted preparation for Goethe and TELC certifications
- **Recorded Lessons**: All classes recorded for review
- **Affordable Pricing**: €350 per course (100 teaching hours)

### Development Phases

- **Phase 1**: Online German courses (A1-B1) for integration, career, and study
- **Phase 2**: Introduction of B2 courses and partnerships with German universities

## Technology Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Modern, responsive design
- Optimized for performance
